# Maternal Risk Monitoring System 🩺🤰

## Overview
Maternal Risk Monitoring is an AI-based web application designed to predict pregnancy risk levels and provide nearby hospital recommendations using geolocation.

## Features
- Risk prediction based on Age, Weight, Symptoms
- AI-based classification
- Geolocation-based nearest hospital suggestion
- Google Maps redirection
- Cloud-ready architecture

## Technologies Used
- Python (Flask)
- HTML, CSS, JavaScript
- Geolocation API
- Machine Learning (Risk Prediction Model)

## How to Run
1. Install Python
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
